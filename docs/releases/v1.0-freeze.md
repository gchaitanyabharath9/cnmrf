# CNMRF v1.0 Frozen Reference Baseline

**Date:** 2026-01-06
**Status:** FROZEN (v1.0.0)

## 1. Declaration
The Cloud-Native Modernization Reference Framework (CNMRF) is hereby declared feature-complete and **frozen at Version 1.0.0**. This establishes an immutable baseline for external review and reference implementation.

CNMRF is an **open-source, platform-neutral architectural reference framework**. It is NOT a product, SaaS, or commercial offering.

## 2. Included Scope (v1.0 Baseline)
The following modules constitute the official v1.0 baseline:
*   **Architecture & Patterns:** Microservices (Spring/NET), Event-Driven, Batch, Storage, 12-Factor, NFRs.
*   **Security & Networking:** Zero Trust, OIDC/OAuth2, mTLS, Network Zoning (Internal/B2B/B2C), API Security.
*   **Governance:** TOGAF ADM Mapping, ARB Operating Model, Standards Catalog, ADRs.
*   **Operations:** CI/CD Pipelines, GitOps, Multi-Env Strategy, FinOps/TCO, Observability.
*   **Migration:** Due Diligence, "6 Rs" Strategies, Data Cutover, Platform Exit Strategy.
*   **Assets:** Reference Code, Diagram Catalog (Visio/UML/ArchiMate), Templates.

## 3. Post-Freeze Exclusions
The following areas are explicitly OUT OF SCOPE for v1.0 and will not be added to this baseline:
*   SaaS-specific business logic (billing, subscriptions).
*   Vendor-specific implementation details beyond illustrative examples.
*   UI/Frontend component libraries.

## 4. Versioning Policy
*   **v1.0.x:** Maintenance only. Non-breaking documentation clarifications, typofixes, or link repairs. No new architectural scope.
*   **v2.0.0:** Reserved for future major architectural changes or breaking governance updates.
