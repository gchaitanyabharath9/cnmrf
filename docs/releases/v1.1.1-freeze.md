# CNMRF v1.1.1 Frozen Reference Baseline

**Date:** 2026-01-06
**Status:** FROZEN (v1.1.1)

## 1. Declaration
CNMRF is hereby declared **frozen at Version 1.1.1**. This release focuses exclusively on IP ownership, content protection, and open-source project governance artifacts.

CNMRF is an **open-source, platform-neutral architectural reference framework**. It is NOT a product, SaaS, or commercial offering.

## 2. Included Scope (v1.1.1 Patch)
*   **IP Protection:** Explicit `COPYRIGHT.md` and `docs/governance/ip-ownership-and-content-protection.md`.
*   **OSS Governance:** `CITATION.cff`, `CONTRIBUTING.md`, `SECURITY.md`, `CODE_OF_CONDUCT.md`.
*   **Attribution:** Updated README citation guidelines.

**No new architectural scope or technical content was added in this patch.**

## 3. Post-Freeze Exclusions
*   SaaS billing logic.
*   Vendor-specific code.
*   UI component libraries.

## 4. Versioning Policy
*   **v1.1.x:** Patches for typos, links, or governance/license clarifications.
*   **v2.0.0:** Major architectural or scope changes.
