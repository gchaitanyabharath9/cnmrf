# Changelog

All notable changes to the Cloud-Native Modernization Reference Framework (CNMRF) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-07
### Added
- **IP Protection**: Enforced strict IP controls including watermarks and copyright footers for authoritative documentation.
- **Standards Compliance**: Added `CITATION.cff` for academic and industry citation support.
- **Zero Trust Patterns**: Expanded architectural reference models to include identity propagation standards.

### Changed
- Standardized documentation structure under `/docs`.
- Updated LICENSE attribution.

## [1.0.0] - 2026-01-01
### Added
- Initial release of CNMRF.
- Core Pillars: Governance, Platform Engineering, Security, Observability.
- Reference implementations for AWS (Helm) and SpringBoot microservices.
